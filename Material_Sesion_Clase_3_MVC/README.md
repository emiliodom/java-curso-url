# Material — Sesión de Clase 3 (Spring Boot primer arranque + MVC mínimo)

Abrir `index.html` en el navegador.

## Objetivo
Lograr un **primer arranque real de Spring Boot** (entorno + primer run + endpoint /hello) y usar **MVC** como mapa mental mínimo para ubicar responsabilidades.

## Archivos
- `index.html`: portada y navegación
- `preparar_pc.html`: Java 17 + PATH + Spring Initializr paso a paso (Windows)
- `clase.html`: guía para impartir la sesión (docente)
- `guion_interactivo.html`: modo clase en vivo (temporizador + checklist + “Decir/Hacer/Chequear”)
- `teacher_oop.html`: material docente extra (OOP básico + estructura del proyecto generado)
- `ejercicios.html`: práctica guiada (entorno, run, endpoint)
- `cheatsheet.html`: hoja de referencia rápida (comandos + MVC mínimo)
- `entregable_equipo.html`: plantilla de evidencias del equipo
- `styles.css`: estilos

> Nota: El material está diseñado para funcionar offline salvo por las librerías CSS/JS que se cargan por CDN.
