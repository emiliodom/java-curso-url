Sesión 3 — Primer arranque de Spring Boot (sin perderse)

En esta sesión hacemos lo difícil, paso a paso: preparar el entorno, generar un proyecto con Spring Initializr, correrlo por primera vez con mvnw y crear un endpoint /hello. MVC se usa solo como mapa mental mínimo (quién hace qué) para organizarse.

Meta de la sesión (resultado visible)
Un proyecto Spring Boot que corre + un endpoint /hello funcionando + una mini-explicación de dónde está el Controller.

Checklist rápido (antes de iniciar)

  • Java 17 instalado (JDK) y verificado con java -version.
  • Editor/IDE listo (VS Code / IntelliJ / Eclipse).
  • Conexión a internet para usar Spring Initializr.
  • MVC: recordar solo “datos / acciones / entrada (Controller)”.
Tip docente: si algo falla, vuelve a lo básico: versión de Java, ruta del proyecto, puerto 8080 y logs.
