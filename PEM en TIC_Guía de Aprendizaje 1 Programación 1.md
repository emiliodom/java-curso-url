**SEGUNDA PARTE: CONSTRUCCIÓN DEL PROYECTO**

**Propósito de la actividad:** Dar los primeros pasos en la programación práctica, escribiendo y ejecutando código Java básico, aplicando buenas prácticas desde el inicio.

**Actividades (Instrucciones):**

1.  **Preparación del Entorno:**
        -   **No instalar nada.** Usar el compilador online: https://www.educative.io/compilers/java
        -   Pegar el template inicial y ejecutar:

                ```java
                public class Main {
                    public static void main(String[] args) {
                        System.out.println("Hola mundo");
                    }
                }
                ```

        -   Nota: En Educative la clase normalmente debe llamarse **Main**.
        -   (Opcional) Si algún estudiante ya tiene Java/IDE instalado, puede usarlo, pero **no es requisito** en la semana 1.

2.  **Práctica de Código (El "Taller de Lógica"):**
    -   Escribir un programa pequeño que imprima sus datos personales en consola.
    -   Crear un programa que realice operaciones matemáticas básicas (suma, resta) usando variables.
    -   Crear un programa con una condición (if/else) y un ciclo (for o while).
    -   Crear al menos un método propio (ej. `saludar()` o `imprimirDatos()`) y llamarlo desde `main`.
    -   **Reto Pedagógico:** Escribir comentarios en el código explicando cada línea como si fuera para un estudiante.

    -   **Trabajo en grupo (entregable):** En equipos de 3–4, elegir 1 reto principal (A o B) y completar además el reto C (depuración). Incluir evidencia de salida (captura o texto) y explicación corta.

3.  **Reflexión Didáctica:**
    -   Escribir un breve párrafo sobre las dificultades encontradas y cómo se podrían abordar en una clase real.

**Recursos:**
-   Guía rápida de uso del compilador online (Educative).
-   Ejercicios de lógica de programación (nivel básico).

**Evaluación (Sumativa):**
*Entrega de los archivos `.java` creados (individual), la solución del reto grupal (A o B + C) con evidencia de salida, y el documento de reflexión didáctica.*
