1.  **Activación de presaberes**

*Actividad: Discusión sobre la reutilización de materiales educativos y cómo los docentes crean "plantillas" para actividades similares. Conexión con el concepto de componentes reutilizables en programación.*

*Tiempo estimado: 15 minutos*

*Recursos: Presentación sobre analogías entre planificación educativa y programación por componentes.*

*Actividad que realizará el estudiante: Compartir experiencias sobre reutilización de planificaciones, identificar elementos repetitivos en sus clases, y reflexionar sobre la eficiencia de tener "plantillas" personalizables.*

2.  **Retroalimentación**

*Actividad: Preguntas y respuestas para introducir React mediante analogías familiares: componentes como plantillas educativas reutilizables.*

*Tiempo estimado: 10 minutos*

*Recursos: Diagramas visuales comparando plantillas de actividades vs componentes React, foro de discusión.*

*Actividad que realizará el estudiante: Formular preguntas sobre cómo se podría aplicar el concepto de "plantillas reutilizables" en el desarrollo web, y entender la conexión entre planificación educativa y programación modular.*

3.  **Construcción del conocimiento**

***Actividad 1: Introducción a React con Analogías Educativas***

*Actividad: Demostración en vivo de React usando analogías educativas. Mostrar cómo crear un componente simple (HeaderClase) comparándolo con una plantilla de encabezado de plan de clase.*

*Tiempo estimado: 40 minutos*

*Recursos: VS Code, React documentación, proyector, ejemplos preparados de componentes educativos.*

*Actividad que realizará el estudiante: Seguir la demostración creando su primer componente HeaderClase, entender JSX como "HTML mejorado", aplicar conceptos de componentes como plantillas reutilizables.*

*Evaluación: Observación del primer componente creado y comprensión de conceptos mediante preguntas dirigidas sobre analogías educativas.*

***Actividad 2: Comparación Práctica - Create React App vs Vite***

*Actividad: Demostración comparativa de configuración de proyectos usando ambas herramientas, explicando ventajas pedagógicas de cada una para diferentes niveles de estudiantes.*

*Tiempo estimado: 35 minutos*

*Recursos: Terminal, Node.js instalado, ambas herramientas configuradas, cronómetro para medir tiempos.*

*Actividad que realizará el estudiante: Configurar un proyecto con la herramienta elegida, documentar diferencias observadas, decidir cuál usar para su plan de clase basado en criterios pedagógicos.*

*Evaluación: Verificación de proyecto configurado correctamente y comprensión de diferencias entre herramientas.*

4.  **Aplicación del conocimiento**

***Actividad 1: Desarrollo de Componentes Educativos***

*Actividad: Creación práctica de componentes React para un plan de clase: ObjetivosCard y ActividadItem, aplicando props para personalizar contenido educativo.*

*Tiempo estimado: 45 minutos*

*Recursos: VS Code, proyecto React configurado, plantillas de componentes proporcionadas, ejemplos de contenido educativo.*

*Actividad que realizará el estudiante: Desarrollar componentes ObjetivosCard y ActividadItem, usar props para personalizar objetivos y actividades, integrar componentes en App.jsx principal.*

*Evaluación: Evaluación formativa de componentes funcionales y uso correcto de props para personalización educativa.*

***Actividad 2: Integración y Presentación del Plan de Clase***

*Actividad: Combinar todos los componentes creados en un plan de clase completo y presentar la aplicación React funcionando.*

*Tiempo estimado: 20 minutos*

*Recursos: Aplicación React desarrollada, proyector para presentaciones.*

*Actividad que realizará el estudiante: Integrar componentes en una aplicación funcional, presentar su plan de clase React, explicar decisiones de diseño y herramientas utilizadas.*

*Evaluación: Evaluación del producto final considerando funcionalidad, aplicabilidad educativa y comprensión de conceptos React.*

5.  **Reflexión sobre mi proceso de aprendizaje**

*Actividad: Reflexión escrita sobre la experiencia de aprender React mediante analogías educativas y la comparación de herramientas de desarrollo. Los estudiantes evalúan cómo los conceptos de componentes se conectan con su práctica docente.*

*Tiempo estimado: 10 minutos*

*Recursos: Plataforma de aula virtual, plantilla de reflexión sobre React y enseñanza.*

*Actividad que realizará el estudiante: Redactar reflexión sobre la conexión entre componentes React y planificación educativa, comparar Create React App vs Vite desde una perspectiva pedagógica, y evaluar aplicabilidad en su futura enseñanza.*

*Evaluación: Evaluación cualitativa basada en la comprensión de conceptos React y su aplicabilidad en contextos educativos reales.*

6.  **Introducción al siguiente tema**

*Actividad: Visualización de videos introductorios sobre React en el mundo real y cómo se usa en aplicaciones educativas modernas.*

*Tiempo estimado: 5 minutos*

*Recursos: Videos seleccionados sobre React en educación y desarrollo web moderno.*

*React para principiantes: https://youtu.be/T_j60n1zgu0*

*Aplicaciones educativas con React: https://youtu.be/bMknfKXIFA8*

*Actividad que realizará el estudiante: Tomar notas sobre aplicaciones reales de React en educación y planificar cómo continuará desarrollando sus habilidades con componentes más avanzados y estado.*

7.  **Actividad de cierre y despedida**

*Actividad: Recapitulación de los puntos clave sobre React, componentes educativos reutilizables, y diferencias entre herramientas de desarrollo. Establecimiento de metas para continuar el desarrollo del plan de clase React.*

*Tiempo estimado: 5 minutos*

*Recursos: Resumen visual de conceptos React aprendidos, checklist de componentes desarrollados.*

*Actividad que realizará el estudiante: Formular preguntas finales, confirmar comprensión de componentes y props, y planificar cómo mejorará su plan de clase React con más funcionalidades y mejor diseño.*
