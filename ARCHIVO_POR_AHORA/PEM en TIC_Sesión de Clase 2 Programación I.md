1.  **Activación de presaberes**

*Actividad: Discusión sobre la evolución del desarrollo web y la importancia de HTML5 semántico en la creación de páginas modernas y accesibles.*

*Tiempo estimado: 15 minutos*

*Recursos: Presentación sobre HTML5 semántico vs HTML tradicional, ejemplos de sitios web modernos.*

*Actividad que realizará el estudiante: Participar en la discusión sobre experiencias previas con HTML y compartir conocimientos sobre elementos semánticos.*

2.  **Retroalimentación**

*Actividad: Preguntas y respuestas para aclarar conceptos sobre HTML5 semántico, CSS3 Flexbox y el papel de la IA en el desarrollo web.*

*Tiempo estimado: 10 minutos*

*Recursos: Foro de discusión en el aula virtual, ejemplos de código HTML5.*

*Actividad que realizará el estudiante: Formular preguntas sobre diferencias entre HTML tradicional y HTML5 semántico, y sobre herramientas de IA para desarrollo.*

3.  **Construcción del conocimiento**

***Actividad 1: HTML5 Semántico con HTML Dog***

*Actividad: Demostración práctica de HTML5 semántico usando HTML Dog (español) como recurso principal. Exploración de elementos como header, nav, main, section, article y footer.*

*Tiempo estimado: 40 minutos*

*Recursos: HTML Dog en español (https://www-htmldog-com.translate.goog/guides/html/), Visual Studio Code, proyector.*

*Actividad que realizará el estudiante: Seguir las guías de HTML Dog en español, crear estructura básica de un plan de clase usando elementos semánticos.*

*Evaluación: Observación de la correcta implementación de elementos HTML5 semánticos durante la práctica guiada.*

***Actividad 2: Introducción a Flexbox Froggy***

*Actividad: Inicio del juego Flexbox Froggy para aprender CSS Flexbox de manera gamificada. Completar los primeros 8 niveles del juego.*

*Tiempo estimado: 35 minutos*

*Recursos: Flexbox Froggy (https://flexboxfroggy.com/), navegadores web.*

*Actividad que realizará el estudiante: Jugar Flexbox Froggy niveles 1-8, documentar las propiedades Flexbox aprendidas (justify-content, align-items, flex-direction).*

*Evaluación: Verificación del progreso en el juego y comprensión de las propiedades Flexbox básicas.*

4.  **Aplicación del conocimiento**

***Actividad 1: Configuración y Uso de GitHub Copilot***

*Actividad: Instalación y configuración de GitHub Copilot en VS Code. Demostración de uso ético y responsable de la herramienta para acelerar el desarrollo web.*

*Tiempo estimado: 25 minutos*

*Recursos: VS Code, extensión GitHub Copilot, cuenta de GitHub.*

*Actividad que realizará el estudiante: Instalar GitHub Copilot, configurar preferencias básicas, practicar con sugerencias simples de HTML y CSS.*

*Evaluación: Verificación de la correcta instalación y comprensión de principios éticos en el uso de IA.*

***Actividad 2: Desarrollo del Plan de Clase con HTML5 y Flexbox***

*Actividad: Inicio del proyecto práctico - creación de una página web de plan de clase usando HTML5 semántico y aplicando Flexbox para el layout.*

*Tiempo estimado: 35 minutos*

*Recursos: VS Code con GitHub Copilot, navegador web para pruebas, estructura base proporcionada.*

*Actividad que realizará el estudiante: Desarrollar estructura HTML5 semántica del plan de clase, aplicar CSS con Flexbox básico, usar GitHub Copilot de manera responsable.*

*Evaluación: Evaluación formativa del progreso en la estructura HTML5 y implementación inicial de Flexbox.*

5.  **Reflexión sobre mi proceso de aprendizaje**

*Actividad: Reflexión escrita sobre la experiencia de aprender HTML5 semántico, Flexbox y el uso de GitHub Copilot. Los estudiantes evaluarán cómo estas herramientas cambian su enfoque al desarrollo web.*

*Tiempo estimado: 10 minutos*

*Recursos: Plataforma de aula virtual, plantilla de reflexión proporcionada.*

*Actividad que realizará el estudiante: Redactar reflexión sobre la diferencia entre HTML tradicional y HTML5 semántico, la utilidad de Flexbox Froggy como herramienta de aprendizaje, y su primera experiencia con IA en programación.*

*Evaluación: Evaluación cualitativa basada en la profundidad de la reflexión y la identificación de beneficios y desafíos de las nuevas herramientas.*

6.  **Introducción al siguiente tema**

*Actividad: Visualización de videos introductorios sobre desarrollo web moderno con HTML5, CSS3 y herramientas de IA.*

*Tiempo estimado: 5 minutos*

*Recursos: Videos seleccionados sobre HTML5 semántico y Flexbox.*

*HTML5 Semántico: https://youtu.be/Q2qOoZiXW3M*

*CSS Flexbox Tutorial: https://youtu.be/3YW65K6LcIA*

*Actividad que realizará el estudiante: Tomar notas sobre los conceptos avanzados presentados y planificar cómo continuará completando Flexbox Froggy y desarrollando su proyecto de plan de clase.*

7.  **Actividad de cierre y despedida**

*Actividad: Recapitulación de los puntos clave sobre HTML5 semántico, primeros pasos en Flexbox, y introducción a GitHub Copilot. Establecimiento de metas para continuar el trabajo fuera de clase.*

*Tiempo estimado: 5 minutos*

*Recursos: Resumen visual de la sesión, recordatorios sobre recursos disponibles (HTML Dog, Flexbox Froggy).*

*Actividad que realizará el estudiante: Formular preguntas finales, confirmar comprensión de las tareas para la siguiente sesión: completar más niveles de Flexbox Froggy y continuar desarrollando el plan de clase con HTML5 semántico.*
