Anotaciones clave

@Controller
@RequestMapping("/recursos")
@GetMapping
@PostMapping
@PathVariable
@ModelAttribute
  • @Controller: clase que maneja vistas (controlador).
  • @RequestMapping: prefijo de rutas.
  • @GetMapping: muestra paginas (GET).
  • @PostMapping: guarda datos (POST).
  • @PathVariable: recibe el id en la URL.
  • @ModelAttribute: llena la entidad desde el formulario.

Imports comunes (Controlador)

import org.springframework.stereotype.Controller;
import org.springframework.ui.Model;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PostMapping;
import org.springframework.web.bind.annotation.RequestMapping;
import org.springframework.web.bind.annotation.PathVariable;
import org.springframework.web.bind.annotation.ModelAttribute;
Cada import habilita una tarea: rutas, datos, vistas.

Dependencias minimas

  • Spring Web
  • Thymeleaf
  • Spring Data JPA
  • BD H2

Rutas sugeridas (CRUD basico)

Accion Ruta Vista/Resultado
Listar GET /recursos recursos/lista
Formulario GET /recursos/nuevo recursos/formulario
Guardar POST /recursos redirect:/recursos

Errores comunes

  • 404: ruta no existe o falta @RequestMapping.
  • TemplateInputException: falta el HTML en plantillas (templates).
  • Failed to configure DataSource: falta dependencia o properties.
  • Null en checkbox: no se manejo available.

Comandos utiles

./mvnw clean
./mvnw spring-boot:run
./mvnw spring-boot:run -Dspring-boot.run.arguments="--debug"
