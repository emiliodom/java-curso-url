Ejercicio 1 — Hola Mundo + 3 líneas

Objetivo: comprobar que compila y entender impresión en consola.

  1. Imprime Hola mundo.
  2. Imprime tu nombre.
  3. Imprime una frase: “Hoy empiezo a programar”.
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    System.out.println("Hola mundo");
    System.out.println("Mi nombre es Ana");
    System.out.println("Hoy empiezo a programar");
  }
}

Ejercicio 2 — Variables “cajitas”

Objetivo: declarar variables y concatenar texto.

  • Crea variables: String nombre, int edad, boolean leGustaProgramar.
  • Imprime todas con un mensaje.
Pista
Para imprimir texto + variable: "Edad: " + edad
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    String nombre = "Ana";
    int edad = 20;
    boolean leGustaProgramar = true;

    System.out.println("Nombre: " + nombre);
    System.out.println("Edad: " + edad);
    System.out.println("¿Le gusta programar?: " + leGustaProgramar);
  }
}

Ejercicio 3 — Operadores matemáticos

Objetivo: practicar +, -, *, /, %.

  • Define int a = 17 y int b = 5.
  • Imprime suma, resta, multiplicación, división (entera) y residuo.
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    int a = 17;
    int b = 5;

    System.out.println("a+b=" + (a + b));
    System.out.println("a-b=" + (a - b));
    System.out.println("a*b=" + (a * b));
    System.out.println("a/b=" + (a / b));
    System.out.println("a%b=" + (a % b));
  }
}

Ejercicio 4 — Prioridad y paréntesis

Objetivo: entender el orden de operaciones.

  • Calcula y muestra: 10 + 2 * 3 y (10 + 2) * 3.
  • Explica (en comentario) por qué cambian.

Ejercicio 5 — Comparación y booleanos

Objetivo: practicar >, <, ==, !=.

  • Declara int edad = 17.
  • Crea booleanos: esMayor (edad >= 18), esMenor (edad < 18).
  • Imprime ambos.

Ejercicio 6 — If/Else: acceso

Objetivo: tomar decisiones.

  • Si edad >= 18 imprime “Acceso permitido”.
  • Si no, imprime “Acceso denegado”.
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    int edad = 17;

    if (edad >= 18) {
      System.out.println("Acceso permitido");
    } else {
      System.out.println("Acceso denegado");
    }
  }
}

Ejercicio 7 — If/Else if: calificación

Objetivo: rangos.

  • Declara int nota = 72.
  • Si nota >= 90: “Excelente”
  • Si nota >= 70: “Aprobado”
  • Si no: “Reprobado”

Ejercicio 8 — Ciclo for: tabla del 5

Objetivo: repetir.

  • Imprime 5×1 hasta 5×10.
Solución sugerida 
public class Main {
  public static void main(String[] args) {
    int n = 5;
    for (int i = 1; i <= 10; i++) {
      System.out.println(n + " x " + i + " = " + (n * i));
    }
  }
}

Ejercicio 9 — Ciclo while: conteo regresivo

Objetivo: practicar condición en repetición.

  • Inicia en 5 y baja hasta 1, luego imprime “¡Despegue!”.

Ejercicio 10 — Método: saludar

Objetivo: modularizar.

  • Crea un método static void saludar(String nombre) que imprima “Hola, …”.
  • Llama al método 2 veces desde main.

Reto final (10–15 min) — Mini programa “Perfil”

Integra variables + if + ciclo + método.

  1. Crea variables de perfil (nombre, edad).
  2. Usa if/else para indicar si es mayor de edad.
  3. Usa un ciclo para imprimir 3 metas del curso (pueden ser textos fijos).
  4. Crea un método imprimirEncabezado() para ordenar la salida.
Ejemplo de estructura (no es la única) 
public class Main {
  public static void main(String[] args) {
    imprimirEncabezado();

    String nombre = "Ana";
    int edad = 20;

    if (edad >= 18) {
      System.out.println("Mayor de edad");
    } else {
      System.out.println("Menor de edad");
    }

    for (int i = 1; i <= 3; i++) {
      System.out.println("Meta " + i + ": practicar todos los días");
    }

    saludar(nombre);
  }

  static void imprimirEncabezado() {
    System.out.println("=== PERFIL ===");
  }

  static void saludar(String nombre) {
    System.out.println("Hola, " + nombre + "!");
  }
}

Trabajo en grupo (25 min en clase) — Retos que hacen pensar

Forma equipos de 3–4. Elijan 1 reto principal (A o B) y completen además el reto C (depuración). Si terminan rápido, tomen una extensión “nivel 2”.

Roles sugeridos (rotar cada 10 min)
  • Driver: escribe el código.
  • Navigator: revisa lógica y lee errores.
  • Tester: propone casos de prueba (valores distintos).
  • Documentador: redacta explicación corta y checklist.
Regla de oro
Cada bloque del programa debe responder: entrada → proceso → salida. Si algo falla, prueben con valores simples.

Construyan un programa que ayude al docente con tareas simples. No requiere entrada del usuario (pueden usar variables fijas).

  1. Declaren variables: String nombre, int edad, int nota1, int nota2, int nota3.
  2. Creen un método static int promedio(int a, int b, int c) y calculen el promedio.
  3. Usen if/else para clasificar el resultado: “Excelente” (>=90), “Aprobado” (>=70), “Reforzar” (<70).
  4. Usen un ciclo (for o while) para imprimir 3 recomendaciones (texto fijo).
  5. Ordenen la salida con un método imprimirEncabezado().
Extensión (nivel 2)
  • Agreguen un método static boolean esMayorDeEdad(int edad) y muestren el resultado.
  • Agreguen validación: si una nota es <0 o >100, mostrar error y no calcular.

Modelen una decisión con reglas claras: salida distinta según combinación de condiciones.

  1. Variables: boolean tareaEntregada, boolean asistio, int puntosExtra.
  2. Reglas (pueden ajustar):
    • Si tareaEntregada && asistio → “Va al día”.
    • Si tareaEntregada && !asistio → “Entregó pero faltó”.
    • Si !tareaEntregada && asistio → “Asistió pero falta tarea”.
    • Si !tareaEntregada && !asistio → “En riesgo”.
  3. Usen un ciclo para simular 3 estudiantes (valores distintos) e imprimir el resultado de cada uno.
  4. Creen un método static String estado(boolean tarea, boolean asistio) que devuelva el texto.
Extensión (nivel 2)
Incorporen puntosExtra: si es >= 2 y “En riesgo”, cambiar a “En observación”.

Copien este código, intenten ejecutarlo y corrijan todos los errores de compilación y lógica.

public class Main {
  public static void main(String[] args) {
    int edad = 18
    if (edad = 18) {
      System.out.println("Mayor de edad");
    } else {
      System.out.println("Menor de edad")
    }

    for (int i = 1; i <= 5; i--) {
      System.out.println("i=" + i);
    }
  }
}
Pista
  • Hay errores de punto y coma.
  • Hay un uso incorrecto de = en una condición.
  • El ciclo tiene un problema de terminación.
Criterio de éxito
El programa debe terminar y mostrar un conteo de 1 a 5.

¿Qué deben entregar?

Usen la plantilla: entregable_equipo.html
  • Código final del reto elegido (A o B) y la solución del Reto C (pueden ser 2 archivos o 2 bloques separados).
  • Captura o texto del output (lo que imprime la consola) para al menos 2 casos distintos.
  • Explicación corta (5–8 líneas): qué hace el programa y cómo lo dividieron en métodos.
  • Checklist de calidad: indentación, nombres claros, comentarios pedagógicos, sin código repetido innecesario.
