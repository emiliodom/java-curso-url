Cheat Sheet

Java básico — lo mínimo que debes recordar

Perfecto para principiantes. Úsalo como referencia rápida.

Template (cópialo siempre)

public class Main {
  public static void main(String[] args) {
    // Tu código aquí
  }
}
En Educative normalmente la clase debe llamarse Main.

Imprimir en consola

System.out.println("Hola");
System.out.println("Edad: " + 20);

If / Else

if (edad >= 18) {
  System.out.println("Mayor");
} else {
  System.out.println("Menor");
}

Ciclos

// for: cuando sabes cuántas veces
for (int i = 1; i <= 5; i++) {
  System.out.println(i);
}

// while: mientras la condición sea verdadera
int x = 3;
while (x > 0) {
  System.out.println(x);
  x--;
}

Métodos

static void saludar(String nombre) {
  System.out.println("Hola, " + nombre);
}
Recuerda llamarlo desde main: saludar("Ana");

Primeros pasos para iniciar en programación (sin instalar)

  1. Ejecuta un Hola Mundo y confirma que compila.
  2. Aprende a leer errores: busca la línea y el mensaje.
  3. Escribe código con estructura: 1) variables 2) proceso 3) salida.
  4. Prueba de a poco: cambia una cosa y ejecuta.
  5. Comenta tu intención: “qué quiero que haga”.
