# Material — Sesión de Clase 1 (Java)

Abrir `index.html` en el navegador.

## Cómo usar el compilador online (sin instalar nada)
- Compilador Java (Educative): https://www.educative.io/compilers/java
- Copiar/pegar el “template” de código desde `cheatsheet.html` o `ejercicios.html`.
- Ejecutar con **Run**.

## Archivos
- `index.html`: portada y navegación
- `clase.html`: guía para impartir la sesión (docente)
- `ejercicios.html`: ejercicios + soluciones sugeridas (en secciones desplegables)
- `cheatsheet.html`: hoja de referencia rápida (Java básico)
- `styles.css`: estilos adicionales

> Nota: El material está diseñado para funcionar offline salvo por las librerías CSS/JS que se cargan por CDN.
