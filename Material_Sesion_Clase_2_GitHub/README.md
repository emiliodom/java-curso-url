# Material — Sesión de Clase 2 (GitHub y control de versiones)

Abrir `index.html` en el navegador.

## Objetivo
Dominar lo esencial de **Git y GitHub** para llevar control de versiones de proyectos educativos.

## Archivos
- `index.html`: portada y navegación
- `clase.html`: guía para impartir la sesión (docente)
- `ejercicios.html`: ejercicios prácticos (VS Code + comandos)
- `cheatsheet.html`: hoja de referencia rápida (Git/GitHub)
- `entregable_equipo.html`: plantilla de entrega del equipo
- `styles.css`: estilos

> Nota: El material está diseñado para funcionar offline salvo por las librerías CSS/JS que se cargan por CDN.
